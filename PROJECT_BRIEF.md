# Portfolio Project Brief

## Purpose

This is Mengqi Cao's UX portfolio: a quiet, editorial site for presenting product
design work, experiments, and case studies with the rhythm of an art book or
design journal.

The site should feel thoughtful, minimal, and publication-like. Typography,
spacing, hierarchy, and restraint matter more than decorative effects.

## Current Direction

- Fixed left navigation and a main editorial content column.
- Pure white background, black primary text, subtle gray metadata, and thin
  borders.
- Landing page with hero statement, Vibe Coding Projects, Featured Works, About,
  and Archive.
- Vibe Coding Projects open as lightweight overlays.
- Featured Works open as dedicated case-study pages.

## Primary Project In Progress

The Council is the most developed case study right now. It uses:

- A horizontal media carousel.
- A two-column overview with testimonial, project metadata, and prototype link.
- Short narrative sections for core problem, introduction, research, ideation,
  prototype walkthrough, and conclusion.

## Working Principle

Do not treat every visual change as a one-off. When a pattern repeats, update the
shared style or component pattern so the whole portfolio stays consistent.

## Current Technical State

The site is currently static HTML, CSS, and JavaScript. A React migration is
planned, but the static version should remain usable while the migration happens
in phases.
